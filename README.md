# Organization-Structure-tree.
ObjectiveTo understand the implementation of Tree data structure in C++.To manage employee hierarchy using nodes.To perform insertion and display operations in a tree.

Types of Organizational Structure
Functional Structure – Departments based on functions like HR, Finance, and Marketing.
Divisional Structure – Organized by product, region, or customer group.
Matrix Structure – Employees report to both functional and project managers.
Hierarchical Structure – Traditional top-to-bottom management system.
Manager Tree
A manager tree represents the hierarchy of managers and employees in an organization.
Example Manager Tree
Plain text
CEO
│
├── HR Manager
│   ├── HR Executive 1
│   └── HR Executive 2
│
├── Project Manager
│   ├── Team Leader
│   │   ├── Developer 1
│   │   └── Developer 2
│
└── Finance Manager
    ├── Accountant 1
    └── Accountant 2

Advantages

Clear reporting system
Easy communication flow
Better task management
